# clicker
🎮 README — Clicker Game
📌 Description

Clicker est un mini-jeu web développé en HTML, CSS et JavaScript.
Le principe est simple : cliquer pour accumuler des points et améliorer ses performances au fil du temps.

Inspiré des jeux incrémentaux (clicker games), ce projet met en avant une logique simple, addictive et évolutive.

🚀 Fonctionnalités
🖱️ Système de clic pour générer des points
📈 Score en temps réel
⚡ Progression rapide et addictive
🎯 Interface simple et accessible
💾 (Optionnel) Sauvegarde locale des données
